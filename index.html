Instructions for Automatic Uploads

Note: This procedure does not work on mobile phones.

  1. Click the Start button.
  2. Click the Copy button or manually select and copy the code.
  3. Log in to your LeetCode account.
  4. Right-click and select Inspect, or press F12.
  5. Go to the Console tab.
  6. To enable pasting, type allow pasting in the console and press Enter.
  7. Paste the copied code and press Enter.
  8. Return to this site to track the upload progress.
